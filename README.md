# PyGUIFolder

## What does it do
Create virtual folders and add files as symlinks to them 

![Image1](images/demo1.png)

## Installation
First clone the repo

```bash
$ git clone https://github.com/n0sys/PyGUIFolder.git
```

Then run the app

```bash
$ python3 app.py
```
